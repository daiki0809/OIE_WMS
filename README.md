# OIE_WMS

## サイト概要
### サイトテーマ
お家で使用する商品の買い忘れ、買い過ぎを防止するアプリ

### テーマを選んだ理由
・いざ必要な時に消耗品がなくなっていると困るため  
・買い物をしている際、すでになくなったであろう商品を購入すると まだ家にあることがよく発生するため

### ターゲットユーザ
お家で消耗品の買い物を担当している方

### 主な利用シーン
どの商品を買わないといけないか確認するとき
